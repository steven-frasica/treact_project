# treact_project

In this project, I recreated a Treact template seen here https://treact.owaiskhan.me/components/landingPages/SaaSProductLandingPage
Using the basics of HTML and CSS learned in this course, I was able to structure the website into ten different sections: the landing consisting of the nav and header, features, quality, steps, values, pricing, testimonials, FAQs, start, and the footer. Using a flexbox structure, each section could be broken up into smaller containers that function as parent elements for smaller elements such as paragraphs, text, images, etc. The flexbox approach allows for robust responsive design, making your website accessible on various screen sizes. 

For sections that weren't required for the project, I used Github Copilot and FES's Duck to answer questions about concepts that weren't covered. For example, Copilot helped build the sliding testimonials, and explained how I could use the `details` and `summary` elements and the built in functionality to hide and reveal the answers in the FAQs.
